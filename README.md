When creating an application using Clean Architecture, there are so many things to create, such as layers, mappings, interfaces, and infrastructure. I want to provide a way to focus solely on business development.

I want to be able to download and register infrastructure from a platform, which will automatically create the interfaces and only require necessary data mapping. This way, developers can concentrate on building and testing the business logic.

I also want the ability to modify the infrastructure functionalities as desired.

Is it possible to change the architecture design according to my preferences? I want to allow developers to improve it freely.

Let's start with a CLI.
